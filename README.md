# ComicsNet

`comicsnet` is a Python package for background subtraction of data cube obtained in ground-based mid-infrared observations.


## Installation

Install directly from the GitHub repository:

``` console
$ python -m pip install git+https://github.com/astronasutarou/comicsnet.git
```

Or download the source code and install it:

``` console
$ git clone https://github.com/astronasutarou/comicsnet.git
$ cd comicsnet
$ python -m pip install .
```
